Projet 01 / 05

Infrastructure Réseau VLAN

Objectif :Comment concevoir une infrastructure réseau adaptée aux besoins d'une entreprise ?
1
Ce projet concerne une entreprise pharmaceutique fictive qui nous a contactés pour mettre en place son infrastructure réseau.
2
On a travaillé sur Cisco Packet Tracer — on a choisi un switch de niveau 3 au cœur du réseau pour gérer le routage, et des switches de niveau 2 dans les étages pour relier les services.
3
On a utilisé une adresse de classe B avec du VLSM, ce qui permet d'adapter la taille des sous-réseaux selon le nombre de machines dans chaque service.
4
Chaque VLAN correspond à un département, donc les services sont isolés entre eux — ce qui sécurise les échanges.
✓ Infrastructure fonctionnelle — VLANs communiquent selon les règles définies

Projet 03 / 05

Intranet HTTPS avec certificat auto-signé

Objectif :Comment sécuriser l'accès à un site intranet en entreprise ?
1
J'ai créé une machine virtuelle avec Ubuntu Server, puis j'ai installé Nginx — c'est le logiciel qui permet d'héberger le site.
2
J'ai utilisé OpenSSL pour créer un certificat de sécurité. Le HTTPS sert à chiffrer les échanges entre le navigateur et le serveur, pour protéger les données.
3
J'ai mis en place une redirection automatique du HTTP vers le HTTPS, pour que personne ne puisse accéder au site en version non sécurisée.
4
L'alerte du navigateur est normale — le certificat est auto-signé, donc il n'est pas reconnu par une autorité officielle, mais le chiffrement fonctionne bien.
✓ Intranet accessible en HTTPS — redirection HTTP automatique fonctionnelle
